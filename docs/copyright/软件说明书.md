# retalk AI编码助手会话管理软件 V1.0

# 用户操作说明书

---

## 第一章 软件概述

### 1.1 软件简介

retalk是一款跨平台的AI编码助手会话管理软件。软件以系统托盘常驻方式运行，用户通过全局快捷键（默认Ctrl+Shift+R）即可唤起Spotlight风格的弹窗界面，快速搜索、浏览和恢复在多种AI编码命令行工具中的历史对话。

### 1.2 适用对象

本软件适用于日常使用AI编码命令行工具（如Claude Code、Codex CLI、Gemini CLI等）进行软件开发的程序员和技术人员。

### 1.3 运行环境

- 操作系统：Windows 10/11、macOS 10.15及以上、Linux
- 硬件要求：x86_64或ARM64处理器，512MB以上内存，100MB以上硬盘空间
- 软件依赖：无需额外安装运行时环境

---

## 第二章 安装与启动

### 2.1 安装

#### Windows
下载`.msi`或`.exe`安装包，双击运行安装向导，按提示完成安装。

#### macOS
下载`.dmg`文件，双击打开后将应用拖入"应用程序"文件夹。

#### Linux
下载`.deb`包使用`dpkg -i`安装，或下载`.AppImage`文件直接运行。

### 2.2 启动

安装完成后，软件自动在系统托盘区域显示图标。可通过以下方式操作：
- 单击托盘图标：显示/隐藏主窗口
- 全局快捷键：Ctrl+Shift+R（Windows/Linux）或Cmd+Shift+R（macOS）

## 第三章 主要功能

### 3.1 会话搜索

在搜索框中输入关键词，软件实时检索所有AI工具的会话记录。支持：
- 中英文混合搜索
- 项目名称搜索
- 项目路径搜索
- 会话内容搜索

### 3.2 会话浏览

会话列表支持两种视图模式：
- **按项目分组**：将会话按所属项目归类显示
- **时间线**：按时间倒序显示，自动分组为"今天"、"昨天"、"本周"、"本月"、"更早"

### 3.3 会话恢复

双击会话条目或按Enter键，软件自动打开终端窗口，切换到对应项目目录，并执行会话恢复命令。

### 3.4 工具筛选

通过工具栏的筛选按钮，可按AI工具类型过滤会话列表，支持：Claude Code、Codex CLI、Gemini CLI、OpenCode、Kilo Code。

### 3.5 收藏与标签

- **收藏**：点击会话条目左侧的星标按钮，收藏的会话始终置顶显示
- **标签**：为会话添加自定义标签（如"bug修复"、"新功能"等），支持按标签搜索
- **自动标签**：在设置中执行自动标签功能，根据首条消息关键词自动识别会话类型

### 3.6 会话预览

使用方向键导航会话列表时，底部预览面板显示选中会话的最近3条消息内容。

### 3.7 右键菜单

右键点击会话条目，弹出快捷操作菜单：
- 新建会话
- 恢复会话
- 在VS Code中打开
- 在文件管理器中打开
- 复制项目路径
- 复制恢复命令
- 导出Markdown
- 导出到文件
- 对比工具

### 3.8 会话对比

当同一项目被多个AI工具使用时，可通过"对比工具"功能并排查看各工具的会话记录。

### 3.9 批量操作

按住Ctrl键点击多个会话条目进入多选模式，可批量导出Markdown。

### 3.10 新建会话

点击搜索栏旁的"+"按钮，选择AI工具和项目目录，快速启动新的编码会话。

## 第四章 生态管理

### 4.1 插件管理

在生态面板的"插件"标签页中，可管理Claude Code和Codex CLI的插件：
- 查看已安装插件
- 浏览可安装插件
- 一键安装/卸载/启用/禁用/更新插件

### 4.2 Skills可视化

"Skills"标签页展示所有已安装的Skills（技能文件），按工具分组显示名称、描述和来源。

### 4.3 MCP服务器管理

"MCP"标签页提供跨工具的MCP（Model Context Protocol）服务器管理：
- 查看各工具已配置的MCP服务器
- 添加新的MCP服务器
- 启用/禁用/移除MCP服务器

### 4.4 工具概览

"概览"标签页显示每个AI编码工具的：
- 安装状态和版本号
- 会话数量、MCP数量、Skills数量
- 数据存储路径
- 版本更新检查
- 未安装工具的一键安装

## 第五章 设置

### 5.1 快捷键设置

点击快捷键输入框后按下组合键，自动录入新的全局快捷键，保存后立即生效。

### 5.2 打开方式

设置双击会话的默认动作：恢复到终端或在VS Code中打开。

### 5.3 终端偏好

选择首选终端程序，支持自动检测和手动选择。支持自定义终端命令模板。

### 5.4 数据更新

配置三种数据更新策略：
- 文件监听：实时监控数据文件变化
- 定时轮询：按设定间隔检查更新
- 按需刷新：打开窗口时检查

### 5.5 主题

支持深色和浅色两种主题切换。

### 5.6 开机自启

设置软件随系统启动自动运行。

## 第六章 使用统计

统计面板展示以下数据：
- 各AI工具的会话数量统计
- 最活跃项目排行（柱状图）
- 月度活跃度趋势（柱状图）
- 频繁活动项目（本周与上周对比）
- 长期未活动项目（30天以上无会话）
