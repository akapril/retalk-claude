## 二、软件功能说明（用于填写"软件的简要说明"）

retalk是一款跨平台的AI编码助手会话管理软件，采用Spotlight风格的弹窗界面，常驻系统托盘，通过全局快捷键快速唤起。软件能够自动扫描和索引用户在Claude Code、Codex CLI、Gemini CLI、OpenCode、Kilo Code等多种AI编码命令行工具中的对话会话数据，提供基于Tantivy引擎的全文搜索功能（支持中文分词），并支持一键恢复历史会话到终端。

主要功能模块包括：
1. 会话管理：支持按项目分组、时间线视图、收藏置顶、标签分类、会话备注、批量导出等功能；
2. 生态面板：提供插件管理（安装/卸载/启用/禁用）、Skills可视化、MCP服务器管理（添加/移除/开关）、工具概览和版本检查等功能；
3. 数据索引：采用Tantivy全文搜索引擎配合jieba中文分词，支持持久化索引和增量同步，实现亚毫秒级搜索响应；
4. 跨平台适配：支持Windows、macOS、Linux三大平台的终端检测、自启动配置和系统集成。

## 三、技术特点（用于填写"技术特点"）

1. 采用Rust语言和Tauri v2框架开发，实现跨平台原生应用，安装包体积约8MB；
2. 使用Tantivy全文搜索引擎（Rust原生Lucene替代方案）结合jieba-rs中文分词库，实现高效中英文混合搜索；
3. 实现三种数据更新策略（文件系统监听、定时轮询、按需刷新），采用增量索引同步机制，保证数据实时性的同时避免性能开销；
4. 采用Provider抽象架构，通过统一的SessionProvider trait实现多种AI编码工具的会话数据解析，支持JSONL、JSON、SQLite三种数据格式；
5. 前端采用原生HTML/CSS/JavaScript，无第三方框架依赖，最小化资源占用；
6. 全部外部进程调用采用异步非阻塞模式（async + spawn_blocking），避免UI冻结。
